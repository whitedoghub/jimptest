<h4><준비></h4>  

- node update (Linux)
```node
> node -v // 현재 버전 확인
> node install -g n // 'n' -> node 버전 관리 모듈 ->  root 계정으로 설치  
> n stable // 안정화 버전 설치. 참조) n latest, n lts, n 11.4.0
> n // node 버전 변경
> n remove 11.4.0 // 특정 버전 삭제 (다른 표기법 : n - 11.4.0)
> n prone // 현재 선택된 버전을 제외하고, 모든 버전 삭제
```
